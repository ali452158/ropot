# ALFA Reports — بوت تداول آلي للمتاجرة في الذهب

<div dir="rtl">

بوت تداول آلي يعمل على منصة MetaTrader 5 (MT5) للمتاجرة في الذهب (XAUUSD)
باستراتيجية **تسليم الأذيل (Wick-to-Wick Rejection)** على فريم M1 مع تنفيذ لحظي
صاروخي.

## الميزات

- ✅ **استراتيجية تسليم الأذيل**: البوت يراقب شموع الذهب على فريم M1 لحظياً،
  وعند اكتشاف ذيل طويل (≥ 50% من حجم الشمعة) ثم عودة السعر لملامسة طرف الذيل،
  يفتح صفقة فوراً.
- ✅ **نمط التردد العالي (HF) — تداول كل شمعة M1**: مفتاح تبديل في لوحة التحكم
  يجعل البوت يفتح صفقة عند كل إغلاق شمعة M1 دون انتظار عودة السعر لطرف الذيل.
  الاتجاه يُحدَّد بـ:
  1. ذيل سفلي طويل → BUY
  2. ذيل علوي طويل → SELL
  3. لا ذيل → زخم الشمعة (close vs open)
  يقلل الفاصل إلى 500ms ليلتقط إغلاق الشمعة فوراً.
- ✅ **إدارة مخاطر صارمة**: TP = 10 نقاط، SL = 7 نقاط، خروج بالوقت بعد دقيقتين.
- ✅ **تنفيذ لحظي**: قرار في أجزاء من الميلي ثانية، تنفيذ على VPS بلندن قريب
  من سيرفرات الوسطاء.
- ✅ **ربط حقيقي مع MT5**: عبر MetaAPI Cloud Cloud API (يدعم MT5 موبايل وكمبيوتر).
- ✅ **نظام اشتراك شهري بأكواد تفعيل**: كل كود يعمل على جهاز واحد فقط لمدة 30 يوماً.
- ✅ **بوت تلجرام لإدارة الأكواد**: المالك فقط يمكنه توليد/عرض/إحصاء الأكواد.
- ✅ **واجهة احترافية باللون الأسود والأبيض والأزرق مع إضاءة نيون**: متجاوبة
  مع الموبايل والكمبيوتر.
- ✅ **لوحة تحكم حية**: سعر لحظي، مخطط شموع يبرز الأذيال الطويلة، إحصائيات
  الأداء، آخر الصفقات.

## آلية العمل

### 1) تفعيل الاشتراك

عند فتح الموقع، يطلب من المستخدم إدخال كود التفعيل. الكود:
- يُولَّد من بوت تلجرام (يملكه مالك النظام).
- يعمل على جهاز واحد فقط (يتم ربطه ببصمة الجهاز عند أول تفعيل).
- صالح لمدة 30 يوماً من أول استخدام.

### 2) تسجيل الدخول إلى MT5

بعد تفعيل الكود، تظهر شاشة تسجيل الدخول إلى MT5:
- **MT5 ID** — رقم الحساب
- **MT5 Password** — كلمة المرور
- **MT5 Server** — اسم السيرفر (مثل ICMarketsSC-Live)

ملاحظة: المستخدم يدخل بيانات MT5 فقط، لا يحتاج لمعرفة أي شيء عن MetaAPI.
كلمة المرور تُخزَّن مشفّرة (scrypt + salt) ولا تُستخدم إلا لربط الحساب وتنفيذ
الصفقات.

### 3) لوحة التحكم

تتضمن:
- اختيار الزوج (فقط أزواج الفوركس + الذهب/الفضة)
- اختيار تايم فريم (M1 افتراضي وموصى به)
- تحديد حجم اللوت
- TP/SL تلقائي (10/7) أو يدوي
- زر تشغيل البوت / إيقاف البوت
- مخطط شموع حي يبرز الأذيال الطويلة بنقاط نيون نابضة
- سعر لحظي (BID/ASK) + السبريد
- إحصائيات الأداء (نسبة النجاح، النقاط، الصفقات المفتوحة)
- قائمة آخر 50 صفقة

## التشغيل على VPS (إنتاج)

### 1) متغيرات البيئة (.env)

```bash
# MetaAPI Cloud — احصل عليها من https://metaapi.cloud
META_API_TOKEN=your-metaapi-cloud-token
META_API_DOMAIN=agiliumtrade.agiliumtrade.ai

# بوت تلجرام — من @BotFather
TELEGRAM_BOT_TOKEN=your-bot-token
ADMIN_TELEGRAM_ID=your-numeric-telegram-id

# عنوان URL للتطبيق (من الـ VPS)
ALFA_APP_BASE_URL=https://alfa-reports.example.com

# سر مشترك بين الموقع وبوت تلجرام (سلسلة عشوائية طويلة)
ADMIN_API_TOKEN=generate-a-long-random-string

# قاعدة البيانات
DATABASE_URL=file:/home/z/my-project/db/custom.db
```

### 2) أوامر التشغيل

```bash
# تثبيت الاعتماديات
bun install

# تهيئة قاعدة البيانات
bun run db:push

# تشغيل الموقع
bun run dev

# تشغيل بوت تلجرام (في نافذة منفصلة)
cd mini-services/telegram-bot
bun install
bun run dev
```

### 3) بوت تلجرام

بعد تشغيل بوت تلجرام، يملك المالك وحده القدرة على:
- `/start` — عرض المساعدة
- `/generate [n]` — توليد n أكواد جديدة (حد أقصى 50)
- `/list` — عرض آخر 20 كود
- `/stats` — إحصائيات سريعة

الأكواد تُولَّد عبر استدعاء آمن لـ `/api/codes/generate` مع رأس `X-Admin-Token`.

## الأمان

- كلمات مرور MT5 لا تُخزَّن كنص صريح، بل مشفّرة بـ scrypt + salt.
- بصمة الجهاز تُحتسب من User-Agent + IP + Accept-Language باستخدام SHA-256.
- أكواد التفعيل مرتبطة ببصمة الجهاز عند أول استخدام، ولا يمكن استخدامها على
  جهاز آخر.
- API توليد الأكواد محمي برأس `X-Admin-Token` المشترك فقط بين الموقع وبوت تلجرام.
- لا يوجد للمستخدم أي وصول مباشر إلى بيانات MetaAPI — كل شيء يتم عبر الخادم.

## الاختبار بدون MetaAPI (وضع المحاكاة)

عند ترك `META_API_TOKEN` فارغاً، يعمل النظام في وضع **SIMULATION**:
- يولّد أسعار ذهب اصطناعية واقعية (مع أذيال طويلة عشوائية).
- ينفّذ نفس منطق البوت بالكامل (تفعيل، دخول، خروج).
- مناسب للتحقق من الواجهة والمنطق قبل النشر على VPS.

## ملاحظات

- النظام يعمل بنفس الكفاءة على MT5 موبايل أو كمبيوتر — لا فرق، لأن الربط
  يتم عبر MetaAPI Cloud.
- لإيقاف البوت: اضغط زر "إيقاف البوت" — سيُغلق أي صفقة مفتوحة بأفضل سعر
  لحظي.
- عند انتهاء صلاحية الكود (بعد 30 يوماً)، يجب تفعيل كود جديد من بوت تلجرام.
